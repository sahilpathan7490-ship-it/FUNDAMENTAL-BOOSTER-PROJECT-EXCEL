# FUNDAMENTAL-BOOSTER-PROJECT-EXCEL# 
📊 Excel FUNDAMENTAL BOOSTER Project
## 📌 Project Overview

This project demonstrates practical Microsoft Excel skills through real-world datasets and business scenarios. It covers student records, sales analysis, employee data management, and advanced Excel functions used in data analysis.

The project is designed to strengthen Excel fundamentals as well as advanced analytical skills required for Data Analyst roles.
